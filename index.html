EZsplit

EZsplit was designed around solving a problem my friends and I experienced quite often. EZsplit allows us to itemize receipts for shared purchases. Some of the features include:

  • an interactive table with receipt information that runs on sockets for live editing
  • a user dashboard that displays metrics on past purchases with a user-feed
  • media queries allow the web app to run nicely on mobile devices
  • Implemented dual-databases to promote security as app scales with more users

For a walkthrough of EZsplit please watch our video. Please give EZsplit a try next time you are dining out with friends and family.
